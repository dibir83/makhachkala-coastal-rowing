# Центр прибрежной гребли · Городской пляж Махачкалы

Проектное предложение для Администрации г. Махачкалы: эллинг 16 × 8 м для хранения лодок и развития
прибрежной (морской) гребли на городском пляже Каспия.

**🔗 Живая ссылка:** https://dibir83.github.io/makhachkala-coastal-rowing/

Одностраничник: открывается на телефоне и компьютере, доступен в любой момент, делается из него PDF
кнопкой **«Скачать PDF»** (правый нижний угол → «Сохранить как PDF»).

## Заменить визуализации на свои (из SketchUp)

Сейчас в блоке «Концепция» стоят временные карточки-заглушки. Чтобы вставить настоящие рендеры:

1. В SketchUp: **File → Export → 2D Graphic…**, формат JPG, ширина ~2000 px.
2. Сохрани два кадра поверх файлов с теми же именами:
   - `images/render-interior.jpg` — вид с лодками внутри эллинга
   - `images/render-exterior.jpg` — эллинг на пляже
3. Обнови сайт:
   ```bash
   cd ~/Desktop/makhachkala-coastal-rowing
   git add images && git commit -m "render: реальные визуализации из SketchUp" && git push
   ```
   Через ~1 минуту живая ссылка обновится автоматически.

> Можешь просто скинуть мне два файла на Рабочий стол — вставлю и опубликую сам.
